# Group Discussion with AI
A web-based AI-powered platform that simulates real-time group discussions with virtual participants to help users improve communication, collaboration, and interview skills.

## Overview
Group Discussion with AI is an intelligent application designed to provide a realistic group discussion environment using artificial intelligence. It allows users to practice discussions, receive feedback, and analyze their performance.
This system bridges the gap between traditional preparation methods and modern AI-driven learning tools by offering an interactive and personalized experience.

## Features
- Real-time group discussion simulation with AI participants
- Voice interaction using speech recognition and synthesis
- AI-generated contextual responses
- Topic selection with difficulty levels (Easy, Medium, Hard)
- Performance scoring and feedback system
- Discussion transcript generation
- Interactive and responsive user interface

## How It Works
1. User enters name, selects a topic, and chooses difficulty level
2. AI generates discussion points and virtual participants
3. User interacts using text or voice
4. AI responds dynamically like real participants
5. System evaluates performance based on participation
6. Final results and score are displayed

## Setup Instructions
Step 1: Clone the repository
git clone https://github.com/your-username/Group_Discussion_with_AI.git
cd Group_Discussion_with_AI

Step 2: Install dependencies
pip install -r requirements.txt

Step 3: Run the application
python app.py

Step 4: Open in browser
http://127.0.0.1:5000/
